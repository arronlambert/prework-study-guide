HTML

  • The !DOCTYPE element defines that this is html
  • The HTML element defines where the HTML format begins
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The style element defines associated CSS styles
  • The base element defines the base URL for a webpage
  • The link element connects an external resource to the HTML document
  • The meta element defines metadata, e.g the charactor set, description, author, keywords and viewpoint
  • h1-h6 define different heading styles
  • li defines listed elements (ul,ol)
